# Code of Conduct

## Our standard

We want JoyBoy to be a welcoming project for contributors of different backgrounds and experience levels.

Please:

- be respectful
- assume good intent
- give actionable feedback
- keep discussions technical and constructive

## Unacceptable behavior

- harassment or insults
- intimidation or hostile conduct
- sharing private information without permission
- bad-faith disruption of discussions or reviews

## Scope

This applies to issues, pull requests, discussions, reviews, and any project-adjacent contributor space maintained by JoyBoy.

Technical disagreement is welcome. Personal attacks are not.

## Reporting

If the problem is a security concern, private token leak, unsafe local file handling, or sensitive content exposure, follow [SECURITY.md](SECURITY.md) instead of opening a public issue.

For conduct concerns, contact the maintainers through GitHub with enough context to understand what happened.

## Enforcement

Project maintainers may remove or moderate comments, issues, pull requests, or contributors whose behavior damages the health of the project.
